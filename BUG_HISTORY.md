## 2026-04-28 - raid.html 파티 선택 오류

### 증상

파티 구성 탭에서 특정 파티 클릭 시:

```txt
ReferenceError: gPresets is not defined
```

### 원인

`renderPartyDetail()` 내부에서 스코프 밖 변수 `gPresets`를 참조함.

### 수정

해당 파티의 preset 정보를 직접 조회하도록 변경.

### 검증

- 파티 선택 시 콘솔 에러 없어야 함
- 여러 레이드/난이도/파티 모두 클릭 테스트 필요
